# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

All participants in the Aratta community are expected to be respectful, constructive, and professional. We are committed to providing a welcoming and inclusive experience for everyone.

For the full text, see https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Enforcement

Instances of unacceptable behavior may be reported via [GitHub Issues](https://github.com/scri-labs/aratta/issues) or by contacting the maintainers directly. All reports will be reviewed and investigated promptly.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
